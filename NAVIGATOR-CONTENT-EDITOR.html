  1. Open NAVIGATOR-DEVMODE.html in a text editor.
  2. Find the CONTENT_DATA object — it starts at approximately line 1715:

  DIRECT:[ ... ],
  EDIT:[ ... ],
  ANIMATE:[ ... ],
  PHOTO:[ ... ],    ← replace this
  DESIGN:[ ... ]     ← and this
};
  3. Select and delete the entire PHOTO:[...] array — from line ~1887 (PHOTO:[) through line ~1912 (the closing ],).
  4. Also select and delete the entire DESIGN:[...] array — from line ~1913 (DESIGN:[) through line ~1997 (the closing ] before };).
  5. Paste the generated code below in the same location (after ANIMATE:[...], and before the closing };).
  6. Save the file and refresh the game in your browser.
  7. Clear localStorage if needed — if you've previously collected placeholder items, open the browser console and run:
